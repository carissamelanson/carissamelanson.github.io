Project Brief

Completing CSS Zen Garden is like a rite of passage for web designers. This project is an example of what can be created only using CSS. My influence for this project was Futurism. I used alot of pseudo classes like :first-letter, :nth-child, :before and :after to bring extra elements to the site.
