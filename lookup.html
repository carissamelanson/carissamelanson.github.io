Project Brief

I created a fully-designed and integrated system for a new web application, Look Up. Look Up is a mentorship app that pairs Austin based designers and developers with mentors and mentees.
View the project live.

Tools Used

HTML, Sass, Bourbon, CSS3 animations and transitions, Reset.scss, Responsive Web Design, jQuery, Photoshop, Illustrator, and Typekit

Process

In two weeks, the project was completed, with research and competitive market analysis, user surveys, style tiles, wireframes, comps, and using Illustrator to create all of the logos and icons for the site, before diving into production in code. Check out out this Style Guide for more details on my process and design elements!
